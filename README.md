Проект Интерпретатор команд для генерации и прохождения лабиринтов

Программа построена по схеме: лексический анализатор (Flex) → синтаксический анализатор (Bison) → исполняющий код на C.

Запуск
1) в консоль вписать команду - make
2) в консоль вписать команду - ./mazez

Доступные команды:

size <высота> <ширина> - Устанавливает размеры лабиринта

choice <1 или 2> - Выбирает тип генерации лабиринта

position <строка> <столбец> - Задаёт начальную позицию игрока

generate или generate walk или generate exit - Создаёт лабиринт (walk - можно пройти самостоятельно, exit - автоматическое прохождение)

help - Выводит справочную информацию

Управление:
WASD - перемещение
q - выход из программы
